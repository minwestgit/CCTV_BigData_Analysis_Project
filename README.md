# CCTV BigData Analysis Project with Apriori Algorithm
대표적인 연관 규칙 분석 알고리즘인 Apriori를 활용한 CCTV 데이터 분석 시스템 <br>
CCTV 데이터 중 동선, 구역, 물품, 회원 데이터를 분석

### DATASET LIST
1. 구역 <br>
시간대별 구역에 방문한 고객 수
2. 동선 <br>
마트 내에서 고객들의 동선 파악
3. 물품 <br>
고객의 물품 터치 횟수 및 구매 여부 <br>
